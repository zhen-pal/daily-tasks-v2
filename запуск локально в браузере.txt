🚀 Запуск приложения
После успешной установки в терминале открытом из папки с файлами введите:
если открыт PowerShell (в шапке), то сначала ввести cmd - переключить на командную строку, затем
npm run dev

В терминале появится:
VITE v5.0.8  ready in 500 ms
➜  Local:   http://localhost:5173/
➜  Network: use --host to expose

Откройте браузер (Chrome или Яндекс.Браузер) и перейдите по ссылке:
http://localhost:5173/

🎉 Приложение должно заработать!
